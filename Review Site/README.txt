This is my website for "Chapman Performance", a fictional automotive performance parts company.

## Features

5 Complete Pages: Home, About, Products, Gallery, and Contact
Responsive Design: Built with Bootstrap 5 for all device sizes
Interactive Elements: JavaScript-powered gallery and contact form
SEO Optimized: Proper meta tags and semantic HTML
Accessibility: Alt text for images, proper contrast ratios
Modern Styling: CSS variables and transitions for smooth effects

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6)
- Bootstrap 5 Framework
- Google Fonts (Roboto)
